# midnight-footnote-privacy

Public hosting for the privacy policy of the personal YouTube upload automation tool
used by the channel [夜更けの寄り道 (@midnight_footnote)](https://www.youtube.com/@midnight_footnote).

- Privacy policy: [PRIVACY.md](PRIVACY.md)
- Terms of Service: [TERMS.md](TERMS.md)

This repository exists solely to publish the privacy policy and terms of service at a stable
public URL (e.g. for the YouTube Data API compliance audit). The channel's source code is kept private.
